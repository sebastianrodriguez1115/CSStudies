# 06 Rails Foundations

## Objetivo

Construir una app Rails simple entendiendo MVC, routing y convenciones.

## Temas

- Estructura de una app Rails.
- MVC.
- Routing REST.
- Controllers.
- Views y layouts.
- Params y strong parameters.
- Helpers.
- Rails console.
- `bin/rails middleware`.

## Practica

- Crear una app Rails.
- Crear recursos REST.
- Inspeccionar rutas.
- Comparar controller Rails contra una app Rack pura.

## Criterio De Cierre

Puedes seguir una request desde Rack hasta controller y response.
