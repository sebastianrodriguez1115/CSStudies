# Grape Reference

## Leer Primero

- Grape: https://github.com/ruby-grape/grape

## Conceptos Clave

- Grape es una app/API Rack.
- El DSL favorece APIs explicitas y versionadas.
- Params y coercion deben vivir cerca del endpoint.
- Serializacion explicita evita filtrar datos internos.
- Swagger/OpenAPI debe salir del contrato real, no de memoria.

## Senales De Dominio

- Puedes montar Grape dentro de Rails.
- Puedes versionar endpoints.
- Puedes manejar errores uniformemente.
- Puedes documentar y testear la API.
