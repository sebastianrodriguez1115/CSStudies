# 01 Ruby Foundations

## Objetivo

Dominar la sintaxis y las estructuras basicas de Ruby.

## Temas

- Variables y constantes.
- Tipos basicos: `String`, `Integer`, `Float`, `Symbol`, `NilClass`, booleans.
- Colecciones: `Array`, `Hash`, `Range`, `Set`.
- Control flow: `if`, `unless`, `case`, `while`, `until`.
- Metodos y argumentos.
- Bloques, `yield`, `Proc` y `lambda`.
- `Enumerable`.

## Practica

- Scripts pequenos para manipular texto y colecciones.
- Transformaciones con `map`, `select`, `reject`, `reduce`, `group_by`.
- Validaciones simples sin framework.

## Criterio De Cierre

Puedes resolver ejercicios de datos con Ruby sin usar Rails ni gems externas.
