## Refactoring Katas Para Despues

- Theatrical Players: https://github.com/emilybache/Theatrical-Players-Refactoring-Kata/tree/main/ruby
- Video Store: ejemplo clasico de `Refactoring`, 1st edition, util para practicar objetos y responsabilidades.
- Gilded Rose: https://github.com/emilybache/GildedRose-Refactoring-Kata
- Tennis: https://github.com/emilybache/Tennis-Refactoring-Kata
- Supermarket Receipt: https://github.com/emilybache/SupermarketReceipt-Refactoring-Kata/tree/main/ruby

Notas:

- `Theatrical Players` esta relacionado con el ejemplo inicial de `Refactoring`, 2nd edition, de Martin Fowler.
- `Video Store` esta relacionado con el ejemplo inicial de `Refactoring`, 1st edition.
- `Gilded Rose`, `Tennis` y `Supermarket Receipt` no son ejercicios de Fowler, pero son buenas katas para practicar refactoring despues.
