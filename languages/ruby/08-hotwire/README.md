# 08 Hotwire

## Objetivo

Agregar interactividad Rails sin convertir todo en SPA.

## Temas

- Turbo Drive.
- Turbo Frames.
- Turbo Streams.
- Stimulus controllers.
- Targets y actions.
- Progressive enhancement.

## Practica

- Actualizar partes de la pagina sin reload completo.
- Crear interacciones pequenas con Stimulus.
- Mantener HTML server-rendered.

## Criterio De Cierre

Puedes decidir cuando usar Turbo, Stimulus o JavaScript normal.
