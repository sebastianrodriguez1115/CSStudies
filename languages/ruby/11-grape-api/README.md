# 11 Grape API

## Objetivo

Construir APIs REST versionadas con Grape dentro de Rails.

## Temas

- `Grape::API`.
- Namespaces y resources.
- Versionado.
- Params, coercion y validaciones.
- Helpers.
- Error handling.
- Integracion con Active Record.
- Auth de API.
- `grape-entity`.
- `grape-swagger`.
- Testing de endpoints.

## Practica

- Montar Grape en Rails.
- Exponer recursos JSON.
- Documentar endpoints.
- Proteger endpoints con token.

## Criterio De Cierre

Puedes crear una API versionada, testeada y documentada sin mezclarla con controllers HTML.
