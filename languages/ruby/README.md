# Programa de Estudio: Ruby, Rack y Ruby on Rails

Este directorio contiene un plan practico para aprender Ruby, Rack, Ruby on Rails, Devise y Grape.

La idea no es acumular teoria. Cada modulo debe cerrar con codigo, tests y una retrospectiva corta.

## Ruta

1. `00-setup`: entorno y herramientas.
2. `01-ruby-foundations`: fundamentos del lenguaje.
3. `02-ruby-oop`: objetos, modulos y diseno Ruby.
4. `03-ruby-stdlib-testing`: libreria estandar y testing.
5. `04-http-and-rack`: HTTP y contrato Rack.
6. `05-rack-middlewares`: middlewares y pipeline Rack.
7. `06-rails-foundations`: Rails MVC y routing.
8. `07-active-record`: modelos, migraciones y queries.
9. `08-hotwire`: Turbo y Stimulus.
10. `09-testing-security`: testing Rails y seguridad.
11. `10-devise-auth`: autenticacion con Devise.
12. `11-grape-api`: APIs con Grape.
13. `12-background-jobs-deployment`: jobs, deploy y produccion.

## Proyectos

- `projects/cli-todo`: Ruby puro.
- `projects/rack-api`: Rack y middlewares.
- `projects/rails-blog`: Rails completo.
- `projects/study-planner-capstone`: proyecto integrador.

## Regla De Trabajo

- Escribe notas cortas, no resumenes largos.
- Cada modulo debe tener ejercicios pequenos y un lab integrador.
- Todo lab debe tener pruebas.
- Si algo parece magico, baja una capa: Ruby, Rack, Rails internals.
