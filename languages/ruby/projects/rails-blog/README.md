# Rails Blog

Proyecto Rails incremental para practicar MVC, Active Record, Hotwire, testing, seguridad y Devise.

## Alcance

- Posts.
- Comments.
- Authors.
- Drafts y published posts.
- Auth con Devise.
- Permisos basicos.
- Hotwire para interacciones pequenas.
- Tests de modelo, request y sistema.

## Regla

Primero funcionalidad simple. Despues seguridad, tests y refactor.
