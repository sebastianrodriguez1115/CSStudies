# Study Planner Capstone

Proyecto integrador para cerrar el programa.

## Idea

Una app para planificar estudio, registrar sesiones, exponer API y automatizar recordatorios.

## Alcance

- Usuarios con Devise.
- Planes de estudio.
- Temas y sesiones.
- Roles basicos.
- API con Grape.
- Docs con Swagger/OpenAPI.
- Jobs con Solid Queue.
- Emails.
- Tests.
- Preparacion para deploy con Kamal.

## Criterio De Cierre

La app debe poder explicarse como producto, como sistema Rails y como API.
