# 02 Ruby OOP

## Objetivo

Modelar comportamiento con clases, objetos y modulos sin sobreingenieria.

## Temas

- Clases e instancias.
- Encapsulamiento.
- `attr_reader`, `attr_writer`, `attr_accessor`.
- Herencia.
- Modulos y mixins.
- Composicion sobre herencia.
- Duck typing.
- Objetos de valor.
- Errores propios.

## Practica

- Modelar entidades pequenas.
- Extraer reglas de negocio a objetos.
- Evitar clases anemicas.

## Criterio De Cierre

Puedes explicar cuando usar clase, modulo, herencia o composicion.
