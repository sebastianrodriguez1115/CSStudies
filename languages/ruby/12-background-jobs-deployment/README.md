# 12 Background Jobs And Deployment

## Objetivo

Llevar una app Rails a preocupaciones reales de produccion.

## Temas

- Active Job.
- Solid Queue.
- Mailers.
- Jobs recurrentes.
- Retries y errores.
- Caching basico.
- Logs y observabilidad basica.
- Docker.
- Kamal.
- Variables de entorno y secrets.

## Practica

- Enviar emails en background.
- Crear job de limpieza.
- Preparar deploy con Kamal.
- Documentar runbook minimo.

## Criterio De Cierre

Puedes correr la app localmente con workers y explicar como se desplegaria.
