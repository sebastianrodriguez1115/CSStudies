# 09 Testing And Security

## Objetivo

Convertir testing y seguridad en parte normal del desarrollo Rails.

## Temas

- Unit tests.
- Request tests.
- System tests.
- Fixtures o factories.
- CSRF.
- XSS.
- SQL injection.
- Strong parameters.
- Sessions y cookies.
- Logging seguro.

## Practica

- Cubrir flujos criticos.
- Probar permisos y errores.
- Auditar inputs y outputs.

## Criterio De Cierre

Puedes demostrar que una feature funciona y que sus casos peligrosos fallan de forma segura.
