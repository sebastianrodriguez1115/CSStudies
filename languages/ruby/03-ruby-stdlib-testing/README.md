# 03 Ruby Stdlib And Testing

## Objetivo

Usar libreria estandar y pruebas para escribir Ruby confiable.

## Temas

- `File`, `Pathname`, `JSON`, `CSV`, `Date`, `Time`.
- `Net::HTTP` y consumo HTTP basico.
- `ERB` para plantillas.
- `Minitest`.
- Fixtures simples.
- RuboCop.

## Practica

- Leer y escribir archivos.
- Consumir una API publica.
- Testear objetos de dominio.
- Automatizar tareas con `Rake`.

## Criterio De Cierre

Puedes crear una gem/app Ruby pequena con tests y tareas reproducibles.
