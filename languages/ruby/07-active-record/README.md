# 07 Active Record

## Objetivo

Modelar datos con Active Record sin perder claridad de dominio.

## Temas

- Migrations.
- Schema.
- Validations.
- Associations.
- Scopes.
- Queries.
- Transactions.
- Callbacks con cautela.
- N+1 queries.

## Practica

- Disenar modelos y relaciones.
- Escribir queries expresivas.
- Medir y corregir N+1.

## Criterio De Cierre

Puedes modelar relaciones comunes y justificar tus validaciones.
